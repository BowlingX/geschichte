# Geschichte Provider

Provides a global state for all namespaces.

<ArgsTable of={Geschichte} />

## Example

You require to define a Toplevel `<Geschichte />` component in your tree to be able
to use any of the provides hooks.


```jsx harmony
import Geschichte from 'geschichte'
import { createBrowserHistory } from 'history'

const history = createBrowserHistory()

export const App = () => (
  <Geschichte history={history}>
   {/* your component tree */}
  </Geschichte>
)
```
